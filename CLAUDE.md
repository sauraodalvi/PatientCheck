# Antigravity Project Guidelines - PatientCheck

## Design & UI/UX
- **Resource Hub**: For any task involving UI design, asset sourcing (fonts, icons, imagery), or prototyping, you **MUST** consult the `@ui-ux-design-resource-hub` skill.
- **Aesthetic**: Maintain the premium "Lumenci AI" look—glassmorphism, dark mode, and high-performance micro-animations (GSAP).
- **Assets**: Favor **Lucide** or **Iconsax** for icons and **Inter/Outfit** for typography.

## Coding Standards
- **Primary Stack**: React, TypeScript, Vite, CSS (Vanilla or Tailwind if specified).
- **Logic**: Favor functional components and clean hooks-based state management.
